# Introduction
Scan Log provides tools for recording activities like restroom cleaning or aisle sweeps. 
- Administrators create sets with the locations of the cleaning activities.
- Staff responsible for completing those activities record the work being done by scanning a unique barcode at the location of the task.
- Staff and/or public reporting obligations are satisfied by the Log View that shows the record of past task completions.

Scan Log allows you to define Sets of tasks. Each set defines one or more locations. For example, you might create a "Restroom Cleaning" set that contains each of the restrooms that require cleaning. 

## Add a new Set
To add a new set to your Scan Log system:
1. Sign into __MyStore__ as a Location Administrator.
2. Select __Scan Log__.
3. Select __Settings__.
4. Click the __＋__ button.
5. Enter a __Name__ for the set. For example, "Restroom Cleaning".
6. Check the __Enable Public Log View__ checkbox to allow anonymous access to your scan log history. Otherwise users must be signed in to MyStore to access logs.
7. Add one or more location with a __Location Name__ and optional __Description__.
8. Click __Create__.

:::tip
To edit an existing set, click the __⋮__ menu on the desired set and select __Edit__.
:::

## Deploying Barcodes
Scan Log records a task when a staff member scans a unique barcode at the location of the task. For example, scanning the barcode on the back of a restroom door to record cleaning that specific restroom.

To deploy barcodes for a set:
1. Click the __⋮__ menu on the desired set and select __Barcode Labels__.
2. Click __Print__.
3. Apply the printed barcodes to the corresponding locations. For example, on the back of a restroom door.

## Staff App
The Staff App is what your staff use to record that they have completed a task.

To access the staff app:
1. Sign into __MyStore__.
2. Select __Scan Log__.

## Log Viewer
The Log Viewer is what your staff, or if enabled, the public, use to view the history of task completions.

To access the log viewer app:
1. Sign into __MyStore__.
2. Select __Overview__.
3. Select the __Log__ icon on the __Scan Log__ card.

If __Enable Public Log View__ is enabled you can access the log viewer app anonymously if you know the URL to it. Typically the public URL would be shared by QR code posted in a location accessible to the intended audience.
